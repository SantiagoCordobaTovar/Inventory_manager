# Inventory Manager (Terminal)

Un sistema sencillo de gestión de inventario en Python usando SQLite.

## Funcionalidades

- Agregar productos
- Mostrar inventario
- Buscar producto por nombre
- Ordenar por categoría
- Exportar a CSV
- Actualizar cantidad o precio
- Eliminar productos

## Requisitos

Python 3.8+

## Cómo ejecutar

```bash
python main.py
```
